Proyectos FUCS

Proyectos desarrollados para la Fundación Universitaria de Ciencias de la Salud (FUCS), optimizando los procesos administrativos de la institución.

Tarjetas de presentación

Aplicación web para generar tarjetas de presentación digitales. Estas pueden ser escaneadas con un teléfono móvil y permiten guardar el contacto del usuario.

Tecnologías
Rúbrica de Entrevistas

Aplicación web para registrar aspirantes a los programas de la institución y asignarles una calificación durante la entrevista de ingreso. Se genera un promedio final para determinar si el candidato es admitido o no.

Tecnologías
Alimentación - Bienestar Universitario

Aplicación web que permite al área de Bienestar Universitario otorgar beneficios alimentarios a los residentes médicos de los programas de posgrado en Medicina.

Tecnologías
Rutas Institucionales

Aplicación web para la reserva de cupos en las rutas institucionales. Estudiantes, docentes o administrativos llenan un formulario y la aplicación guarda la reserva del asiento.

Tecnologías
Prácticas Empresariales

Aplicación web que permite a las facultades asignar entrevistas a los practicantes y gestionar los escenarios de práctica.

Tecnologías
Facturación Electrónica

Conector que extrae datos de OracleSQL y los almacena en una base de datos MariaDB para ser consumidos por un servicio web de un proveedor.
